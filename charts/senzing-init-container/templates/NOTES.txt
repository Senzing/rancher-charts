## Overview

A job to perform all of the Senzing initializations which require root permission.
The initializations are seen in the init-container.py script.

More information:

- [GitHub](https://github.com/Senzing/docker-init-container)
- [Helm Chart](https://github.com/Senzing/charts/tree/master/charts/senzing-init-container)

## Command

The command being run:

```console
init-container.py
  {{- if .Values.senzing.debug }} \
  --debug "{{ .Values.senzing.debug }}"
  {{- end }}
  {{- if .Values.senzing.databaseUrl }}
  --database-url {{ .Values.senzing.databaseUrl }}
  {{- end }}
  {{- if .Values.senzing.engineConfigurationJson }}
  --engine-configuration-json {{ .Values.senzing.engineConfigurationJson }}
  {{- end }}
```
